Generar casos de prueba
Genera casos de uso JSON a partir de la definicion Swagger del ambiente, manteniendo una estructura clara para su revision, ejecucion y exportacion en Postman.

1. Elegi el formato

Legacy XML Queda disponible para el camino SOAP anterior, pero ya no es el foco principal.
JSON Usa la definicion Swagger del ambiente para armar requests REST y colecciones Postman.

2. Elegi el destino

Postman Genera una colección Postman SOAP lista para descargar desde la app.
Luego SOAP Base preparada para un runner o exportador dedicado.

3. Usar el ambiente actual

Todavia no hay configuracion cargada desde el wizard.
Podes pegar la URL del Swagger UI o directamente la del JSON OpenAPI. El builder va a intentar resolverla automaticamente.

4. Constructor de cadenas

Busca servicios, agregalos con clic o arrastrandolos al lienzo y configura cada paso desde el panel lateral antes de exportar el JSON.

<!-- ABRE DATOS DEL MÉTODO -->
::: note Método para validar la existencia de una persona a partir de país y documento.
Este servicio permite invocar el método **PublicPersons.validateExistence**.

**Nombre publicación:** PublicPersons.validateExistence

**Programa:** [Pendiente de completar]

**Alcance:** Global
:::
<!-- CIERRA DATOS DEL MÉTODO -->

<!-- ABRE TABLA DE DATOS -->
::: tabs #Datos

@tab Parámetros de Entrada

Nombre | Tipo | Comentarios
:--------- | :----------- | :-----------
countryId | Integer | Identificador del país del documento.
documentTypeId | Integer | Tipo de documento.
documentNumber | String | Número de documento.
@tab Datos de Salida

Nombre | Tipo | Comentarios
:--------- | :----------- | :-----------
exists | Boolean | Indica si la persona existe (true/false).
@tab Errores

Código | Descripción
:--------- | :-----------
10002 | Error en la ejecución del programa.
10011 | Sesión inválida.

:::
<!-- CIERRA TABLA DE DATOS -->
